Detailed Description

Graph representation of the network.

This class creates a graph data structure based on a topology Message. It can be used to find shortest paths between AttachmentPoints(SwitchPorts).

Author
Andre Kutzleb
Date
15.07.2015
Remarks
TODO this graph structure is very inefficient.(each SwitchPort is a node) if performance is too weak, consider optimizing it (Andre)
